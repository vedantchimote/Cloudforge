- URL: http://localhost:8200
- Token: `dev-token` (development only)

### Store Secrets
```bash
vault kv put secret/cloudforge/database \
  username=cloudforge \
  password=secret123
```

### Read Secrets
```bash
vault kv get secret/cloudforge/database
```
